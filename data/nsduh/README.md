# NSDUH Data

Download the 2024 NSDUH Public Use File (.RData format) from:
https://www.samhsa.gov/data/data-we-collect/nsduh-national-survey-drug-use-and-health/datafiles/2024

Place `nsduh_analysis_ready.RData` in the parent `NSDUH Data/` directory.
This file is generated by the NSDUH analysis pipeline (scripts 01-02 in `NSDUH Data/R/`).
